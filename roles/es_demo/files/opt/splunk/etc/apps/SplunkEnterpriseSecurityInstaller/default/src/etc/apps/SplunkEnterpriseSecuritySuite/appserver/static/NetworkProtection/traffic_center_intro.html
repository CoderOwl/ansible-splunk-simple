Introduction

The Traffic Center dashboard profiles overall network traffic, helps you detect trends in type and changes in volume of traffic, and lets you isolate the cause (for example, a particular device or source) of those changes. This helps you determine when a traffic increase is a security issue and when it is due to an unrelated problem with a server or other device on your network.

Use the filtering options at the top of the screen to limit which items are shown. You can configure new data inputs through Splunk Manager, or search for particular network intrusion events directly through Incident Review. Clicking chart elements or table rows on this page will take you to the raw events that are represented.
