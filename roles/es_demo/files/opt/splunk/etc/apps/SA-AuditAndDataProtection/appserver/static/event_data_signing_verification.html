Verifying Data Integrity Using IT Data Signing

To verify data that is digitally signed, select the "show source" action on the raw event. This will cause Splunk to display all of the source data for the given block. The top of the view source page will display a message indicating if the source data was validated or not. No message will be displayed if data signing is disabled.

See Splunk's wiki for more information.
